<h1 align="center">Tungsten</h1>
<br/>
<p><b>Another command line tool to handle Roblox assets similar to Tarmac and Asphalt.</b></p>
<p>Designed mainly for images, Tungsten mimics Asphalt and Tarmac closely, with spritesheet packing that Asphalt lacks.</p>
<p>Originally built for <a href="https://github.com/notmagniill/LucideRoblox">LucideRoblox</a>.</p>
<h2>Features</h2>
<ul>
  <li>🔄️ Syncing images to Roblox</li>
  <li>⚙️ Generates Luau code for you to reference them in game</li>
  <li>🎯 Targets Roblox users and groups</li>
</ul>
<h2>Goals</h2>
<ul>
  <li>Support <b>more asset types</b> similar to Asphalt (Sounds, Models, Animations etc.)</li>
  <li>Ability to process SVGs to PNGs</li>
  <li>Defining assets you already uploaded</li>
  <li>High DPI detection of images similar to Tarmac</li>
  <li>More features that my brain can think of in the future</li>
</ul>
