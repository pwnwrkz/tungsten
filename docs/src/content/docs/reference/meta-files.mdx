Meta files (`.tmeta`) allow you to customize the metadata of your assets, such as their names and descriptions—on the Roblox website.

## Full example

```txt
name = "Tungsten Meta Files"
description = "Look I have a custom description!"
```

## Field Reference

| Field         | Type     | Description                                                   |
| ------------- | -------- | ------------------------------------------------------------- |
| `name`        | `string` | The display name of the asset or spritesheet on Roblox.       |
| `description` | `string` | The description text that appears on the asset's Roblox page. |
